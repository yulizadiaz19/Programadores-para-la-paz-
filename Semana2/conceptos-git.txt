REPOSITORIO:Un repositorio es un espacio donde se almacenan todos los archivos y carpetas de un proyecto. También guarda el historial de cambios realizados durante el desarrollo. Los repositorios permiten organizar el trabajo y facilitar la colaboración entre programadores utilizando herramientas como Git y GitHub.
COMMIT:Un commit es una acción que guarda oficialmente los cambios realizados en un proyecto. Cada commit funciona como un punto de guardado en el historial del proyecto. Además, permite regresar a versiones anteriores si ocurre algún error o problema durante el desarrollo.
PUSH:El comando push se utiliza para enviar los cambios guardados localmente hacia un repositorio remoto, por ejemplo en GitHub. Esto permite actualizar el proyecto en internet, compartir avances con otras personas y mantener una copia de seguridad del trabajo realizado.
